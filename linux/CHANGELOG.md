# Changelog - Alfredo Tools

## v0.9 (2026-07-03)
### Añadido
- Módulo `resumen-ejecutivo` - Resumen completo del sistema
- Módulo `memoria` - Información detallada de memoria
- Módulo `procesador` - Información detallada del CPU
- Módulo `discos` - Información detallada de discos con SMART
- Módulo `temp-discos` - Monitoreo de temperatura de discos
- Módulo `temp-discos-simple` - Temperatura de discos (simple)
- Módulo `temp-cpu` - Monitoreo de temperatura de CPU
- Módulo `fix-hdmi` - Solución de problemas HDMI
- Módulo `screen-fix-quick` - Solución rápida pantalla negra
- Módulo `monitor-screen` - Monitoreo de cambios en pantallas

### Mejorado
- Menú de ayuda completo y organizado por categorías
- Módulo `salud` con información detallada de CPU, memoria y discos
- Módulo `analizar-logs` con filtro temporal
- Soporte para NVMe con temperatura y SMART

### Corregido
- Detección de temperatura en discos NVMe
- Información de caché del procesador
- Formato de salida en módulos
