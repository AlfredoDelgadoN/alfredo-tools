# Alfredo Tools - Versión Python (Windows nativo)

Herramienta de diagnóstico para Windows sin necesidad de WSL.

## 📦 Módulos

| Módulo | Descripción | Comando |
|--------|-------------|---------|
| `salud` | Estado del sistema | `python alfredo.py salud` |
| `speed` | Velocidad de Internet | `python alfredo.py speed` |
| `discos` | Información de discos | `python alfredo.py discos` |

## 🚀 Instalación

```cmd
git clone https://github.com/AlfredoDelgadoN/alfredo-tools-python.git
cd alfredo-tools-python
pip install -r requirements.txt
